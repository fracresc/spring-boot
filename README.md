## Stock Exchange

## Usage

- Run `mvn clean package`
- Inside the target folder of cli, run the `cli-1.0-SNAPSHOT-jar-with-dependencies.jar` jar with an argument
    - E.g.: `java -jar cli-1.0-SNAPSHOT-jar-with-dependencies.jar AA`
